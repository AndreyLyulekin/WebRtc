WebRTC: ручной сигналинг, оффер/ансвер уже с кандидатами

Для двустороннего видео/аудио нажмите Get Media на обеих сторонах до обмена оффер/ансвер. (Можно и после, но потребуется повторная переговорка для добавления треков.)
Local
Remote
Отправитель (инициатор)
  1. (опц.) Get Media
  2. Create Offer → дождись ICE: complete → Copy Offer и отправь другу
  3. Вставь его Answer → Apply Answer
Получатель
  1. Вставь Offer от друга → Accept & Create Answer
  2. Дождись ICE: complete → Copy Answer и отправь ему
Подсказки
  • Кнопки «Copy …» активируются только при ICE: complete — это гарантирует кандидатов внутри SDP.
  • Если на мобиле нет звука — нажмите Enable remote audio.
