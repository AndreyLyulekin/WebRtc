1) Отправитель → создаёт Offer и передаёт получателю

1) Получатель ← вставляет Offer и создаёт Answer
Памятка
  • Шаг A (Отправитель): Get Media (по желанию) → Create Offer → Copy Offer → отправить другу.
  • Шаг B (Получатель): вставить Offer → Accept & Create Answer → Copy Answer → отправить обратно.
  • Шаг C (Отправитель): вставить Answer → Apply Answer.
  • Чат включится, когда откроется DataChannel.
Без TURN иногда не соединится за жёсткими NAT — для друзей обычно ОК.
